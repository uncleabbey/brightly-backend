Welcome to Brightly Backend API

About The Project

An e-learning platform that will support children’s education in the most effective way, to cushion the effects of school closures caused by the coronavirus outbreak.

With this product, learners can take their daily video lessons, have group interactions with their teachers and peers and do their daily class activities as they would in a traditional classroom.

Target Users

Kids: age 6 - 10 years, their parents and teachers/guardian

Running Locally

To Run the api locally, Follow the following steps

  • Clone or fork the repo and cd into the
  • type `npm install` or `yarn` to install packages.
  • create .env file in the root folder. and the following

    SEC_KEY=string
    DEV_DB_URL=mongodb://localhost/blog-dev
    NODE_ENV=development

  • After that run `npm run dev`
